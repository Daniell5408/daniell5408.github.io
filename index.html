🛠 How It Works

ArduManager bridges the gap between GitHub and your Arduino IDE. Usually, you must download zip files, extract them, and move them to folders. ArduManager handles this list for you.

  • Sync: The software reads your current library list.
  • Update: It finds newer versions of your tools on GitHub automatically.
  • Install: It places files in the correct folder so your Arduino IDE sees them right away.

📋 System Requirements

Ensure your computer meets these needs for the best experience:

  • Operating System: Windows 10 or higher.
  • Software: Arduino IDE 1.8.x or 2.x installed.
  • Network: Active internet connection to reach GitHub.
  • Storage: At least 100MB of free disk space.

💡 Common Questions

Do I need an account? No. You do not need a GitHub account to run the tool.

Will this break my existing libraries? No. ArduManager copies files into folders. It does not overwrite files unless you tell it to update a specific library.

Does it work with portable IDE versions? Yes. You can point the software to your custom folder path during the first launch.

🔍 Troubleshooting

If the software does not find your Arduino IDE, check your settings tab. You can manually select the path to your "Arduino" folder. Most users find this folder inside their "Documents" directory.

If you see an error about network access, check your firewall. The software requires permission to talk to GitHub. If the problem continues, restart your computer and try again.
